# Models

This directory stores downloaded/cached models.

## Embedding Models

When you run embedding-based scoring, sentence-transformers will automatically download models here:
- `all-MiniLM-L6-v2/` (~90MB)
- `e5-mistral-7b-instruct/` (~14GB)
